# Add a display warning icon to your visual

In a Power BI visual, a display warning icon can notify the user of a possible problem or error 

Display warning icon is an ability to notify the user of a possible problem\ error that  occurred in visual. 

The purpose of the warning icon is to present to user that something happened, maybe some unexpected error, but at the same time the visual itself may still be rendered and functional.  

 

The warning icon will be displayed with customized title and details: 

The title will appear when mouse hovers on the icon and as the title of warning dialog 

The detailed text will only be displayed within the warning dialog. 

## Limitations

The following limitations refer to the display icon text.

* The maximum length of the title is 70 characters

* The maximum length of the text is 2,000 characters

* The text is only displayed as plain text
